# FlyGram

FlyGram — это Telegram Web K в формате удобного Android-приложения.

Здесь сохранён привычный интерфейс веб-версии Telegram, но добавлены мобильные улучшения, чтобы пользоваться им было комфортно каждый день.

## Что вы получаете

- Оригинальный интерфейс Telegram Web K, без лишних изменений в стиле чатов.
- Уведомления о новых сообщениях, включая работу в фоне.
- Поддержка прокси: HTTP, SOCKS5 и MTProto.
- Быстрое добавление прокси по ссылкам `tg://proxy` и `https://t.me/proxy`.
- Кэширование чатов и медиа для более стабильной и быстрой работы.
- Доступ к ранее загруженному контенту без интернета.
- Раздел загрузок с управлением файлами внутри приложения.
- Удобный выбор фото, видео и документов через системные Android-окна.
- Жесты и навигация, адаптированные под мобильное использование.
- Настройки внешнего вида и поведения интерфейса.
- Адаптация цветов под тему и общее оформление FlyGram.

## Для кого это приложение

FlyGram подойдёт тем, кто любит Telegram Web K и хочет пользоваться им на Android как полноценным мобильным клиентом.

## Системные требования

- Android 9.0 и выше

## Важно

FlyGram — неофициальный клиент.
